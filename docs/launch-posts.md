## LinkedIn

🚀 Shipped an addon for @bradygaster's Squad — the multi-agent team framework for GitHub Copilot.

The problem: setting up a new Squad takes time. Interview agents, write charters, configure routing. Great for tailored teams. Slow for rapid POCs.

Snap Squad gets you a working Squad in ~15 seconds:

```
npx snap-squad init I need a fast team for a hackathon
```

One command. Describe what you need in plain English. You get a full team with CoreAI builder skills built in as defaults — routing rules, agent charters, MCP tool configs, all from real project experience.

4 presets: generalist, speed, mentor, specialist. Your next Copilot session is already squad-aware.

Open source. Built by its own squad. Goal: contribute upstream.

🔗 https://github.com/paulyuk/snap-squad
